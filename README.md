# 弹幕漂移

一个静态 Canvas 网页小游戏。玩家控制绿色主角球，在逐步增强的弹幕中尽可能存活更久，拾取道具改变局势，并争取更高分。

## 运行

可以直接用浏览器打开 `index.html`。本地测试建议运行：

```bash
node server.mjs
```

然后打开：

```text
http://127.0.0.1:4173
```

## 操作

- 移动：`WASD` 或方向键
- 开始 / 重新开始：`Space` 或页面按钮
- 暂停 / 继续：`P` 或暂停按钮
- 桌面全屏：点击页面上的全屏按钮进入，全屏后可用右上角按钮或 `Esc` 退出
- 手机端：横屏默认进入游戏全屏布局；竖屏显示规则说明和横屏提示
- 触控 / 指针：在场地内拖动

## 道具

- `C`：清除场上敌对球
- `S`：短时间免疫碰撞
- `L`：大幅减速敌对球

时效道具会在主角球附近显示紧凑数字倒计时，数字颜色与道具球颜色一致。

## 语言

顶部 `EN` / `中文` 可以切换英文和中文。游戏会在当前浏览器中记住上次选择。
